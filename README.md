# 🧮 Projeto 1: Suíte de Testes Computacionais

Repositório dedicado à primeira atividade prática da disciplina de **Engenharia de Software** do curso de Ciência da Computação na **UEPB (2026.1)**. O objetivo deste projeto é aplicar os conceitos fundamentais de testes unitários utilizando o ecossistema moderno do JavaScript.

---

## 🛠️ Tecnologias e Ferramentas

O projeto foi estruturado utilizando as seguintes ferramentas de desenvolvimento e qualidade:

* **Runtime:** [Node.js](https://nodejs.org/) (LTS v20+)
* **Engine de Testes:** [Vitest](https://vitest.dev/) 
* **Qualidade de Código:** [ESLint](https://eslint.org/) 
* **Formatação:** [Prettier](https://prettier.io/) 
* **Histórico:** Git & GitHub com mensagens de commits semânticos
